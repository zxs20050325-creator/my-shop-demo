# 冀遗筑梦 - 河北非物质文化遗产电商平台

这是一个展示和销售河北非物质文化遗产产品的电商平台，采用前后端分离架构部署。

## 项目结构

- `backend`: 后端 Node.js 服务，提供 RESTful API
- `frontend`: 前端静态页面，实现用户交互界面

## 功能特性

- 非遗产品展示
- 用户注册登录
- 购物车功能
- 订单管理
- 数据持久化存储

## 部署

本项目已配置为在 Render 平台上自动部署，前后端分离部署以提高性能和可扩展性。

### 后端部署

- 使用 Node.js + Express 架构
- 通过 Supabase 连接数据库
- 提供 RESTful API 服务

### 前端部署

- 静态文件托管
- 通过 API 与后端交互
- 响应式设计，支持多种设备

## 技术栈

- 后端: Node.js, Express, Supabase
- 前端: HTML, CSS, JavaScript
- 部署: Render (前后端分离部署)

## 项目特色

- 专注于河北非物质文化遗产产品的展示和销售
- 采用新中式极简设计风格，融合传统文化与现代交互
- 主色调为青黛色（#2F4842）和鎏金色（#C1A268）
- 提供完善的用户交互体验，包括水墨光标、3D倾斜效果等

## 非遗商品类别

- 蔚县剪纸
- 沧州铁狮子
- 曲阳定窑
- 唐山皮影
- 武强年画
- 衡水内画
- 易水古砚
- 花丝镶嵌
- 玉田泥塑
- 抚宁吹歌

## API接口

后端提供以下API接口：
- 用户注册/登录
- 商品列表查询
- 购物车管理
- 收藏功能
- 订单处理
- 管理员数据统计

## 部署配置

项目使用render.yaml配置文件进行部署，分别部署后端API服务和前端静态网站服务。
