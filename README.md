# 冀遗筑梦 - 河北非物质文化遗产电商平台

这是一个展示和销售河北非物质文化遗产产品的电商平台，采用前后端分离架构部署。

## 项目结构

- `backend`: 后端 Node.js 服务，提供 RESTful API
- `frontend`: 前端静态页面，实现用户交互界面

## 功能特性

- 非遗产品展示
- 用户注册登录
- 购物车功能
- 订单管理
- 数据持久化存储

## 部署

本项目已配置为在 Render 平台上自动部署，前后端分离部署以提高性能和可扩展性。

### 后端部署

- 使用 Node.js + Express 架构
- 通过 Supabase 连接数据库
- 提供 RESTful API 服务

### 前端部署

- 静态文件托管
- 通过 API 与后端交互
- 响应式设计，支持多种设备

## 技术栈

- 后端: Node.js, Express, Supabase
- 前端: HTML, CSS, JavaScript
- 部署: Render (前后端分离部署)
